# Design Optimization Guidelines for AI Everywhere Brand

## 1. Visual Brand Guidelines
* **Brand Aesthetic**: Neo-Futuristic Minimalist
* **Visual Voice**: Sophisticated, seamless, and omnipresent. The design should feel like it's "breathing" through the digital layer.
* **Design Philosophy**: Use "Digital Veins" (glowing lines) to connect organic human elements to the background. Ample negative space to prevent clutter.

---

## 2. Typographic Hierarchy
| Font Role | Canva Font Recommendation | Suggested Sizing / Weight | Use Case |
| :--- | :--- | :--- | :--- |
| **Primary Header** | League Spartan | Extra Bold, 72pt+ | Main title, hooks, big statements |
| **Sub-Header** | Montserrat | Semi-Bold, 36pt | Secondary information, captions |
| **Body Text** | Inter | Regular, 18pt | Explanatory text, lists, details |

* **Typography Readability Rule**: Use high-contrast colors (White/Cyan on Dark) to ensure text pops against the digital background.

---

## 3. Custom Color Palette
| Color Role | Color Name | HEX Code | Distribution (60-30-10 Rule) | Best Used For |
| :--- | :--- | :--- | :--- | :--- |
| Dominant (60%) | Midnight Obsidian | `#020617` | Backgrounds, deep space | Core background space |
| Secondary (30%)| Deep Cobalt | `#1E293B` | Secondary elements, gradients | Subtle depth and texture |
| Accent (10%) | Electric Cyan | `#22D3EE` | Glowing lines, key text, icons | Call outs, critical metrics, button highlights |
| Extra Shade | Slate Mist | `#94A3B8` | Secondary text, fine lines | Detail and structure |

---

## 4. Layout & Grid Blueprint
* **The Grid**: Use a 1080 x 1350 px (LinkedIn Portrait) canvas. Apply a 4-column grid. Position the main focal point (the human/AI interaction) in the center-right intersection.
* **Focal Point**: The glowing "Digital Vein" or connection point between a human element and the digital environment.
* **Whitespace Rule**: Keep at least 150px of padding from all edges. Do not allow text to touch any graphic elements.

---

## 5. Canva Step-by-Step Optimization Workflow
1. **Template Search Queries**:
   - Search for templates matching: `"cyberpunk minimalist"` or `"tech abstract gradient"`
   - Search for elements matching: `"glowing network line"` or `"holographic interface"`
2. **Setup and Creation**:
   - Create a custom size of 1080 x 1350 px.
   - Apply the custom Color Palette codes under Canva's Brand Kit or Document Colors.
3. **Power-User Canva Shortcuts**:
   - `Press 'R' for rectangles (to create background containers)`
   - `Use 'Position' -> 'Layers' to manage depth between human and AI elements`
4. **Recommended Export Settings**:
   - **Format**: `PNG (Lossless)`
   - **Quality/Size**: `2x size for maximum crispness on mobile screens`
